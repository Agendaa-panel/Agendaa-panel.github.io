Multi-AI · несколько независимых ИИ-агентов

У каждого агента — свой API-ключ, провайдер (Groq / OpenAI / Anthropic / Custom), своя модель и своя задача. Можно запустить всех одной общей задачей или дать каждому отдельную. Ключи и история хранятся только в памяти браузера.

⚡
Компьютер без задержки

Поддерживаемые провайдеры: Groq (api.groq.com), OpenAI (api.openai.com), Anthropic (api.anthropic.com), а также любой Custom OpenAI-совместимый эндпоинт. Файл полностью статический, работает прямо из браузера, без сервера.
